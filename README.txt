PARA CONFIGURAR
- vá em <pasta do projeto>/application/config, e procure o arquivo "config.php";
- Dentro do arquivo modifica o $config['base_url'] para a url do projeto;
- Salva e fecha o arquivo;
- Agora iremos configurar o banco de dados, dentro da pasta, procure o arquivo "database.php";
- Dentro do arquivo modifique o 'hostname', o 'username', 'password' se tiver senha e o 'database';
- Salva e fecha o arquivo;
- Agora abre o navegador e digite a url do projeto, pronto o projeto está configurado;
- Para acessar digite o email: agenor@test.com.br e a senha: 123;

- Senha padrão quando cria um fornecedor ou um usuário novo: 123;
